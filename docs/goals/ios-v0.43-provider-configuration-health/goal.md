# Foil iOS v0.43 Provider Configuration Health

## Outcome

Make provider setup and provider failure states understandable and recoverable
for closed beta testers without exposing credentials.

## Oracle

This board is complete when sanitized receipts prove configured, missing,
invalid, network-failed, transcription-failed, and recovered provider states,
and no logs, receipts, docs, or UI screenshots expose provider secrets.

## Starter Command

`/goal Follow docs/goals/ios-v0.43-provider-configuration-health/goal.md.`
