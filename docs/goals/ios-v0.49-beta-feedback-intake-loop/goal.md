# Foil iOS v0.49 Beta Feedback Intake Loop

## Outcome

Give closed beta testers a privacy-safe, structured way to report setup,
recording, insertion, recovery, and target-app friction.

## Oracle

This board is complete when tester handoff, issue templates, labels or triage
docs, and receipt examples collect the minimum actionable beta feedback without
requesting private transcripts, screenshots, contacts, messages, email content,
account names, phone numbers, or credentials.

## Starter Command

`/goal Follow docs/goals/ios-v0.49-beta-feedback-intake-loop/goal.md.`
